## 📖 ᴅᴇꜱᴄʀɪᴘᴛɪᴏɴ
StellarPhone adalah plugin smartphone lengkap untuk Minecraft dengan sistem:

- ✅ SMS System (kirim pesan ke player lain)
- ✅ Call System (telepon antar player)
- ✅ GPS System (teleport ke lokasi tersimpan)
- ✅ Group Chat (chat grup dengan @pesan)
- ✅ Notes (catatan pribadi)
- ✅ Playlist (putar musik in-game)
- ✅ Gadget (senter, kompas, kamera, timer)
- ✅ Battery System (baterai & pengecasan)
- ✅ Backup & Restore (data tersimpan)
- ✅ Radio (broadcast ke semua player)
- ✅ Toko HP (beli smartphone & charger)
- ✅ Vault Economy Support

## 📥 ɪɴꜱᴛᴀʟʟᴀᴛɪᴏɴ
1. Download `StellarPhone.jar` dari [Releases](../../releases)
2. Copy ke folder `plugins/` server Minecraft
3. Restart server
4. Konfigurasi di `plugins/StellarPhone/config.yml`

## 🔌 ᴅᴇᴘᴇɴᴅᴇɴᴄɪᴇꜱ
| Plugin | Keterangan |
|--------|------------|
| [Vault](https://www.spigotmc.org/resources/vault.34315/) | Economy support (opsional) |

> **Note:** Vault bersifat opsional. Plugin tetap berjalan tanpa Vault, namun fitur toko HP tidak akan berfungsi.

## 🎮 ᴄᴏᴍᴍᴀɴᴅꜱ
| Command | Description | Permission |
|---------|-------------|------------|
| `/hp` | Buka menu HP | `smartphone.use` |
| `/givehp` | Berikan smartphone ke player | `smartphone.admin` |
| `/tokohp` | Buka toko HP | `smartphone.use` |
| `/setvillage` | Simpan lokasi village | `smartphone.use` |
| `/angkat` | Angkat panggilan masuk | `smartphone.use` |
| `/tolak` | Tolak panggilan masuk | `smartphone.use` |
| `/tutuptelepon` | Akhiri panggilan aktif | `smartphone.use` |
| `/radio <pesan>` | Broadcast ke semua player | `smartphone.use` |
| `/groupchat <create/join/leave/list>` | Kelola group chat | `smartphone.use` |
| `/backup <save/list/restore>` | Backup & restore data | `smartphone.use` |
| `/pin <set/unlock>` | Atur PIN pengaman | `smartphone.use` |
| `/sync` | Sinkronisasi data | `smartphone.use` |

## 🔋 ʙᴀᴛᴛᴇʀʏ ꜱʏꜱᴛᴇᴍ
- Baterai berkurang saat fitur aktif (senter, kompas, musik)
- Cas HP dengan item **Charger** (klik kanan)
- Battery Saver Mode hemat 50% konsumsi baterai
- Notifikasi saat baterai lemah
